# IDS Alert Investigation (Snort)

## Scenario

In this investigation, a PCAP file containing network traffic was analyzed using an Intrusion Detection System (IDS). The objective was to detect suspicious activity and identify a machine attempting to connect to the target system via SSH.

## Tools Used

* Snort IDS
* Linux Command Line
* PCAP Network Traffic File

## Command Used

```
sudo snort -q -l /var/log/snort -r Task.pcap -A console -c /etc/snort/snort.conf
```

## Command Breakdown

* `sudo` → Runs Snort with administrative privileges.
* `-q` → Quiet mode (suppresses startup messages).
* `-l /var/log/snort` → Directory where Snort logs are stored.
* `-r Task.pcap` → Reads packets from the PCAP file instead of monitoring live traffic.
* `-A console` → Displays alerts directly in the terminal.
* `-c /etc/snort/snort.conf` → Loads the Snort configuration and rule set.

## Investigation Process

1. Navigated to the directory containing the PCAP file.
2. Executed Snort with the appropriate flags to analyze the PCAP traffic.
3. Snort applied its detection rules to the packets inside the capture file.
4. Alerts generated by Snort were displayed in the console output.
5. Examined the alert information to identify suspicious network activity.

## Findings

The IDS detected an SSH connection attempt directed toward the target system. By reviewing the packet information and alert output, the source IP address responsible for the connection attempt was identified.

## Key Concepts Learned

* Role of Intrusion Detection Systems in network monitoring.
* How Snort detects attacks using rule-based signatures.
* Difference between analyzing live traffic and analyzing PCAP files.
* Identifying SSH traffic during network investigations.

## Conclusion

Intrusion Detection Systems such as Snort can analyze network traffic to detect potential attacks. By applying detection rules to captured packet data, analysts can identify suspicious behavior such as unauthorized SSH connection attempts and investigate the source of the activity.
